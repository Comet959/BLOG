课程分为三个部分：
--------------

1. LLM基本原理：
    1.1 数学：线性代数、微积分、概率统计
    1.2 Python：基本语法、数据分析库、数据预处理、机器学习库
    1.3 神经网络：基本原理、训练和优化、过拟合、实现多层感知机
    1.4 NLP：文本预处理、特征提取技术、单词嵌入、循环神经网络RNN

2. LLM科技：
    2.1 LLM架构：高级视图、序列化、注意力机制、文本生成
    2.2 构建指令集：Alpaca-like数据集、高级科技、过滤数据、提示词模板
    2.3 预训练模型：数据管道、因果语言建模、缩放法、高性能计算
    2.4 监督微调：全面微调、LoRA、QLoRA、Axolotl、DeepSpeed
    2.5 从人类反馈中强化学习：偏好数据集、近端策略优化、直接偏好优化
    2.6 评估：传统指标、一般基准、特定任务基准、人类评估
    2.7 量化：基本技术、GGUF和llama.cpp、GPTQ和EXL2、AWQ
    2.8 新趋势：位置嵌入、模型合并、专家的混合、多模态模型

3. LLM工程：
    3.1 运行LLM：LLM APIs、开源LLMs、提示词工程、结构输出
    3.2 构建矢量存储：摄取文件、拆分文档、嵌入模型、向量数据库
    3.3 检索增强生成：编排器、Retrievers、记忆、评估
    3.4 高级RAG：查询构建、Agent工具、后期处理、编程LLM
    3.5 推理优化：Flash注意力、键值缓存、投机解码
    3.6 部署LLM：本地部署、demo部署、服务器部署、边缘部署
    3.7 安全LLM：提示词黑客攻击、后门、防御措施
